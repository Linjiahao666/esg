# ESG 测试数据文件说明

本目录包含用于测试数据导入功能的 CSV 文件。这些文件使用了各种常见的中文字段名，可以用于测试智能字段匹配功能。

## 文件列表

### 1. employees_sap.csv - 员工数据

模拟 SAP HR 系统导出的员工数据格式，包含：

- 员工编号、姓名、性别、部门、职位等基本信息
- 入职/离职日期、在职状态
- 政治面貌、工会成员、残疾人、少数民族等特殊标识
- 共 12 条测试数据

### 2. energy_consumption.csv - 能源消耗数据

包含各类能源的消耗记录：

- 电力（常规电网 + 光伏发电）
- 天然气、水、柴油、汽油
- 按月统计，包含费用和设施信息
- 共 16 条测试数据

### 3. suppliers.csv - 供应商数据

供应商管理数据：

- 供应商编号、名称、分类
- 所在地、是否本地供应商
- 合同金额、ESG 评级
- 共 12 条测试数据

### 4. training_records.csv - 培训记录

员工培训数据：

- 安全培训、技能培训、合规培训、ESG 培训等
- 培训时长、费用、机构、方式
- 考核结果
- 共 14 条测试数据

### 5. safety_incidents.csv - 安全事故记录

工伤和安全事故数据：

- 工伤、未遂事件、财产损失、环境事故
- 严重程度分级
- 伤亡人数、损失工时、经济损失
- 共 7 条测试数据

### 6. donations.csv - 公益捐赠

企业社会责任捐赠数据：

- 现金捐赠、物资捐赠、志愿服务
- 教育、扶贫、救灾、环保、健康等领域
- 共 10 条测试数据

### 7. waste_data.csv - 废弃物数据

废物管理数据：

- 危险废物、一般废物、可回收物
- 数量、处置方式
- 共 16 条测试数据

### 8. carbon_emissions.csv - 碳排放数据

温室气体排放数据：

- 范围一：直接排放（固定燃烧、移动燃烧）
- 范围二：间接排放（外购电力，含光伏抵扣）
- 范围三：其他间接排放（商务差旅、员工通勤）
- 共 13 条测试数据

## 使用方法

1. 在系统中选择对应的数据源（如"员工数据"）
2. 点击"导入数据"按钮
3. 选择对应的 CSV 文件
4. 系统会自动进行智能字段匹配
5. 确认映射关系后执行导入

## 字段映射测试

这些文件使用了与标准模板不同的字段名称，可以测试以下智能匹配能力：

- **精确匹配**：如"姓名" → name
- **别名匹配**：如"员工编号" → employeeNo
- **模糊匹配**：如"所属部门" → department
- **值转换**：如"男/女" → male/female，"是/否" → true/false
